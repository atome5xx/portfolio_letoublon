Compétences Ressources

L'entreprise voulait lancer une application web pour automatiser et faciliter la gestion des clients pour les avocats. Ils avaient besoin d'une base de données pour stocker les différentes information.

J'ai d'abords réalisé plusieurs réunions avec les différents membres associés au projet. Ainsi qu'une analyse des premiers documents créer avant mon arrivé pour initier la création de la base de données.

J'ai ensuite commencé à créer un modèle relationel grâce à l'outils Mysql Workbench.

Après validation du MVC par l'entreprise, j'ai intégré la base de données sur leur hébergeur OVH grâce au script ".sql" générer par mysql workbench et à l'interface phpmyadmin intégrer à ovh.
On m'a ensuite fournis les droits pour créer des administrateurs de la base de données avec différents droits ainsi que la gestions des sauvegardes.

En mission secondaire, j'ai également créer des documents explicatifs sur "utiliser git", "comprendre un modèle MVC", "comprendre la base de données créer"...
En vu de la future application web, j'ai créé la connexion en php à la base de données ainsi que les premières requète SQL.
