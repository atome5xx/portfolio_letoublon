(projet de cours de SLAM (option programmation de mon BTS), nous devions créer un site qui utilise et puisse mettre à jour une base de données.)

Compétences Ressources

J'ai commencé par créer mon site avec HTML et CSS. J'ai choisi de le faire sur les chats avec l'utilisation de frame.

J'ai créé ma base de données (ci-contre) grâce aux commandes de MySQL. J'ai ensuite rempli ses tables.

Connexion de mon site web à ma base de données grâce au PHP.
Depuis mon site on peut interroger ma base de données.

Des formulaires ont été mis en place permettant aux utilisateurs de remplir la base de données avec l'aide du php.
L'utilisateur peut créer, modifier ou supprimer son profil grâce à leur clé primaire donnée lors de la création du profil.

La plupart des pages sont générées avec le PHP. Pour la création de page, le PHP fait appel à la base de données car pour afficher la liste des 53 races de chats, il est bien plus simple de faire une fonction plutôt que de devoir les rentrer à la main.
